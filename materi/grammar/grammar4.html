Grammar 10: Simple Past Tense in English

The **Simple Past Tense** is used to describe completed actions in the past. Regular verbs add **-ed** to form the past tense, while irregular verbs have different forms. Example: "She **visited** her grandmother last weekend." / "They **went** to the park yesterday." The negative form is made using **did not (didn't) + base verb**. Example: "He **did not eat** breakfast this morning." Questions are formed using **Did + subject + base verb?** Example: "Did she call you yesterday?"

Grammar 11: Past Continuous Tense in English

The **Past Continuous Tense** is used to describe actions that were happening at a specific moment in the past. It is formed using **was/were + verb-ing**. Example: "She **was reading** a book when I called her." / "They **were playing** football at 5 PM." The negative form is made using **was not (wasn't) / were not (weren't) + verb-ing**. Example: "He **wasn't watching** TV when I arrived." Questions are formed using **Was/Were + subject + verb-ing?** Example: "Were you sleeping at midnight?"

Grammar 12: Past Perfect Tense in English

The **Past Perfect Tense** is used to describe an action that was completed before another action in the past. It is formed using **had + past participle**. Example: "She **had finished** her homework before dinner." / "They **had left** before I arrived." The negative form is made using **had not (hadn't) + past participle**. Example: "He **hadn't seen** the movie before yesterday." Questions are formed using **Had + subject + past participle?** Example: "Had you ever traveled abroad before last year?"
